## ❓ БЫСТРЫЕ ОТВЕТЫ
**Почему Docker?** → Изоляция сервисов, одинаковое окружение dev/prod, легкий деплой  
**Как работает CI/CD?** → Push → Tests → Build → Deploy → Notify  
**Что если сервер упал?** → Prometheus алерты + автоматический рестарт контейнеров  
**Почему микросервисы?** → Масштабируемость, независимый деплой каждого сервиса  
**Как мониторите?** → Prometheus собирает метрики → Grafana визуализирует
