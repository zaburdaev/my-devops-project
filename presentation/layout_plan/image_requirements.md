# Image Requirements

All visual content in this presentation is rendered as:
- **Coded components** (architecture diagrams, pipeline flows, tech card grids)
- **Charts** (Recharts bar chart on Slide 10)

No photographs, illustrations, or external image assets are needed. All visuals are fully implementable in Next.js + Tailwind CSS code.
